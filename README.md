# FastAPI Full-Stack Application (Blogging Website) - WIP

This repository contains a learning project built with FastAPI that explores how to build a full‑stack web application with both a JSON API and server‑rendered HTML pages.
It covers backend fundamentals like routing, data validation, database integration, authentication, and authorization, along with a basic frontend for interacting with the API.
